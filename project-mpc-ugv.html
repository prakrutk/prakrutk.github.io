MPC for UGVs in Unknown Environments

Status: Completed Course: Foundation of Robotics Tools: Python, CasADi
GitHub

Abstract

Course project implementing Model Predictive Control (MPC) for Unmanned Ground Vehicles (UGV) navigating unknown environments with obstacle avoidance. The system generates optimal waypoints and controls in real-time.

Methodology

  • System model: Bicycle model with kinematic constraints
  • Cost function: Minimizes tracking error and control effort
  • Constraints: Collision avoidance using safety margins
  • Solver: Sequential Quadratic Programming (SQP)

Results

Successfully demonstrated obstacle avoidance in static and dynamic environments with replanning frequency of 10 Hz.
