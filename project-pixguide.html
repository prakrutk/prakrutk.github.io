PixGuide Navigation Device

Status: Completed (Published Patent) Lab: IvLabs, VNIT Nagpur Advisor: Prof. Shital Chiddarwar Tools: Arduino, C++, GPS, OLED, Bluetooth
GitHub 10 stars · 1 fork · C++

Abstract

PixGuide is a hardware navigation device for two-wheelers designed to help riders navigate without mounting phones or expensive GPS devices on their vehicles. The device provides turn-by-turn directions via an intuitive LED matrix display.

The main motive is to help two-wheelers navigate without having to use different applications or mounting mobile phones on their vehicles. The device is compact and works in bright sunlight as well as in the dark.

This project resulted in a published patent (CBR Ref. 201921049473, 2018) for an innovative navigation system using voice pack technology.

Innovation

The key innovation is using Google Maps voice pack frequencies to decode navigation commands. Instead of expensive Google APIs, the device listens to audio cues from Google Maps and maps them to visual directions on the LED display.

  • No custom app required
  • Works with existing Google Maps
  • Extremely low cost compared to commercial solutions

Hardware Components

  • Arduino Pro Mini: Main microcontroller
  • MAX7219 8x8 LED Matrix: Direction display
  • GPS Module: Location tracking
  • Bluetooth Audio Receiver: Receives voice commands from phone
  • Li-ion Battery: Portable power supply
  • Operational Amplifier: Signal processing

Mechanical Design

  • Garmin Quarter Mount: Compatible with all cycle/motorcycle handlebars
  • Sun Hood: Sliding mechanism for daylight visibility
  • Torch Slot: Front mount for visibility at night
  • Water-resistant: Weatherproof enclosure

Features

  • Compact design: Fits on handlebar or helmet
  • GPS integration: Reads location from GPS module
  • Direction indicators: LED arrows for turn directions
  • Bluetooth connectivity: Receives route from smartphone app
  • Low power: Runs on rechargeable battery
  • Day/Night visibility: Clear display in all lighting conditions

Impact

Provides an affordable and safe navigation solution for two-wheeler riders in India, reducing the need for phone mounting and improving road safety. Can also be extended for four-wheeled vehicles and used as a cycling direction indicator.
