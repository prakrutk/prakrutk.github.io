PixGuide Navigation Device

Status: Completed Lab: IvLabs Advisor: Prof. Shital Chiddarwar Tools: Arduino, C++, GPS, OLED

Abstract

PixGuide is a hardware navigation device for two-wheelers designed to help riders navigate without mounting phones or expensive GPS devices on their vehicles. The device provides turn-by-turn directions via an intuitive display.

Features

  • Compact design: Fits on handlebar or helmet
  • GPS integration: Reads location from GPS module
  • Direction indicators: LED arrows for turn directions
  • Bluetooth connectivity: Receives route from smartphone app
  • Low power: Runs on rechargeable battery

Impact

Provides an affordable and safe navigation solution for two-wheeler riders in India, reducing the need for phone mounting and improving road safety.
